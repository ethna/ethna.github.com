メールを送信する


$attach の指定方法: 
  • 既存のファイルを添付するとき
  
    1.   array('filename' => '/tmp/hoge.xls', 'content-type' => 'application/vnd.ms-excel')
      2. 

  • 文字列に名前を付けて添付するとき
  
    1.   array('name' => 'foo.txt', 'content' => 'this is foo.')
      2. 

  'content-type' 省略時は 'application/octet-stream' となる。   複数添付するときは上の配列を添字0から始まるふつうの配列に入れる。
